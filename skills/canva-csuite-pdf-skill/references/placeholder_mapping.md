# Placeholder Mapping

## Required sections
- `title`
- `executive_summary`
- `strategic_priorities`
- `risk_matrix`
- `citations`

## Optional sections
- `annexes`
- `audience`
- `tone`
- `content`

## Binding contract
Each mapped section provides at least:
- `left`
- `top`
- `width`
- `font_size`

Mappings live in `assets/template_placeholders.json` and are copied into runtime jobs by `build_canva_runtime_job.py`.
